SurpriseHaiku - Discovering Unusual Haikus On Twitter

 2014-08-20  No tag
SurpriseHaiku was an experiment that parsed random twitter tweets to see if they followed the Haiku cadence of 5 / 7 / 5 (syllables). I ran this experiment during the 2014 Olympics to try and focus around Olympics related tweets. This was an application that I built to learn more about the twitter API and dip my toes in the world of Natural Language Parsing or NLP. While the application is not running at this point, you can check out the historical tweets on the twitter profile.
